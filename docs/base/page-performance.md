## 异步加载的方式
1. 动态脚本加载；
2. defer；
3. async
    
## 异步加载的区别：
    
- defer是在HTML解析完之后才会执行，如果是多个，按照加载的顺序依次执行
- async是在加载完之后立即执行，如果是多个，执行顺序和加载顺便无关
